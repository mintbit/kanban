# Rails + Hotwire Kanban

This is a very basic Kanban implementation demonstrating how to use Hotwire in a Rails project.

**Note:** Requires Rails 6.1 or higher

## Features

- Columns/lists
- Cards
- Attachments
- Comments
- Tasks

## Installation

1. bundle install
2. yarn install
3. rails db:setup
4. rails s
5. bin/webpack-dev-server
